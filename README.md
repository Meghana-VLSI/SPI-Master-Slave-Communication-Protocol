# SPI Master-Slave Communication using Verilog

## Overview

This project implements the **Serial Peripheral Interface (SPI) Master-Slave Communication Protocol** using Verilog. The design demonstrates synchronous serial communication between a master and a slave device through the four standard SPI signals: **MOSI**, **MISO**, **SCLK**, and **Chip Select (CS)**. The project includes RTL design, simulation, synthesis, and functional verification using Vivado.

---

## Features

- SPI Master implementation
- SPI Slave implementation
- 8-bit serial data transmission
- Full-duplex communication
- FSM-based communication control
- RTL design using Verilog
- Functional simulation
- RTL schematic generation
- Synthesis using Vivado

---

## Project Objectives

## SPI Signals

| Signal | Description |
|---------|-------------|
| MOSI | Master Out Slave In |
| MISO | Master In Slave Out |
| SCLK | Serial Clock generated by Master |
| CS | Chip Select |
