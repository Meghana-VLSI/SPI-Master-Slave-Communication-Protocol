# RTL Code

## Overview

This folder contains the Register Transfer Level (RTL) Verilog source files for the SPI Master-Slave Communication project.

## Description

The RTL design implements the complete SPI communication protocol using synthesizable Verilog. The design consists of separate modules for the SPI Master and SPI Slave, enabling reliable serial communication through the standard SPI interface.

Each module is developed using modular design principles, making the project easy to understand, simulate, synthesize, and maintain.

## RTL Modules

- SPI Master
- SPI Slave
- Top Module (if applicable)

## Design Features

- 8-bit serial communication
- Full-duplex data transfer
- Shift register implementation
- Serial clock generation
- Chip Select control
- FSM-based communication
- Synthesizable Verilog

## Files

- spi_master.v
- spi_slave.v
- spi_top.v 

These files contain the complete RTL implementation of the SPI communication system.
